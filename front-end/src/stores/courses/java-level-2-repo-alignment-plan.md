## Plan

2. Expand `java-level-2.ts` to surface any missing file I/O, error handling, and data-structure work that belongs in the public course.
3. Collapse pure reference or solution folders into notes instead of duplicating them as separate first-class projects.
4. Decide whether threading belongs as a preview addendum or should remain outside the core Java 2 path.
