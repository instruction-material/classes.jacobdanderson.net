# Python Level 3 Repo Alignment Note

## Current Status

Resolved on 2026-05-14 as part of the course source naming and structure
cleanup.

`instruction-material/Python-Level-3` now uses the common project wrapper shape:
each active project folder has `README.md`, `starter/`, and `solution/`.

Legacy top-level starter folders that had matching base projects were folded
into the base project `starter/` directories. Course links now target those
role directories directly.

## Remaining Policy

Folders under `_archived-unlinked/` are retained as source-history material and
should not be treated as active course projects unless they are restored and
linked from the course site in the same change.
